# Resultados — uma pasta por ferramenta

Coloque aqui as imagens geradas. Exemplo:

```text
results/midjourney/
  T1-taverna.webp
  T2-tony-retrato.webp
  T3-tony-consistencia.webp
  T4-balao-unico-pt.webp
  T5-dialogo-dois-pt.webp
  notas.md
```

Copie [[_template/notas|notas.md]] para cada pasta nova.
